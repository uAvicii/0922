从零实现一个简洁博客页面

发布于 2025-09-20 · 6 min read

搭建个人博客并不一定需要复杂的框架。本页采用原生 HTML/CSS/JS，足以满足大部分静态内容展示需求。

布局

使用 CSS Grid 构建主布局，内容区 + 侧边栏；为侧边栏开启 position: sticky，在滚动时保持目录可见。

暗色模式

通过 data-theme + CSS 变量切换主题，用户选择持久化在 localStorage。

const THEME_KEY = 'site-theme';
const preferred = localStorage.getItem(THEME_KEY) || 'light';
document.documentElement.setAttribute('data-theme', preferred);

可访问性

按钮具备可读的 aria-label；颜色对比尽可能符合 WCAG 推荐标准。

学习笔记：小而美的原则

发布于 2025-09-15 · 4 min read

保持页面结构简单、样式清晰、交互克制，能显著降低维护成本。

命名与可读性

变量与函数命名尽量表达意图；拆分复杂逻辑；避免深层嵌套。

组件化思维

即便在纯 HTML 里，也可以用 卡片、按钮、标签 这些可复用的视觉单元思考问题。
